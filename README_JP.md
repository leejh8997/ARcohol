<div style="text-align:left;">
<h2 style="border-bottom: 1px solid #d8dee4; color: #282d33;">💡 プロジェクト紹介 </h2>
<div>Flutterベースのカクテルレシピ/コマースアプリで、マイバー(MyBar)機能とARデモンストレーション、レビューおよび注文まで可能なオールインワンプラットフォームです。</div>
<div>ユーザーは、自分が持っている材料をベースにレシピを推薦してもらい、製品の購入やレビューを作成することができます。</div>
<div>既存のカクテルアプリのほとんどが外国語基盤であることを考慮し、英語レシピをハングルに翻訳·精製して韓国のユーザーに馴染みのあるUI/UXで実現しました。</div>
<div>Firestoreを利用したデータベース設計とFlutter UIを通じた直観的なユーザー経験を提供するアプリです。</div>
</div>

<div style="text-align:left;">
<h2 style="border-bottom: 1px solid #d8dee4; color: #282d33;">🎇プロジェクトレビュー</h2>
<ul>
<li>今回のプロジェクトは、チームメンバー各自が担当した領域に責任感を持って協業して完成したFlutterベースのカクテルレシピ/コマースアプリです。</li>
<li>企画段階からFirestore NoSQL構造設計、Flutter UI/UX実装、AR機能連動、決済·レビューシステム実装まで、全般的なアプリ開発の全過程をチーム内の役割分担を通じて効率的に進めることができました。</li>
<li>また、様々な状態管理と条件付きレンダリング処理、ポートワン決済連動、ユーザーの利便性を考慮した直観的なUI構成など、実戦レベルの機能を実現し、アプリ開発に対する経験を広げることができました。</li>
<li>Firebaseベースの認証およびデータの流れ、ARとUnityの連動、クローリングおよびハングル化処理など、多方面の技術を適用し、それぞれの技術力も共に成長することができたプロジェクトでした。</li>
<li>チーム全体が主導的に貢献し、共に成長した経験だったという点で意味が深かったです。</li>
</ul>
